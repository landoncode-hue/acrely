Key Directories Summary:
========================

1. apps/ - Contains both mobile (React Native/Expo) and web (Next.js) applications
2. packages/ - Shared packages (config, services, ui components, utilities)
3. scripts/ - Deployment and verification scripts for all systems
4. supabase/ - Database migrations, edge functions, and configuration
5. tests/ - End-to-end, UAT, and unit tests
6. docs/ - Technical documentation

Total Files: ~200+ files across the codebase
Total Lines of Code: ~50,000+ lines
