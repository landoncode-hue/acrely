### Practice Scenarios

#### Scenario 1: New Customer Walk-in
**Situation:** Customer walks in interested in purchasing a plot.

**Tasks:**
1. Create customer profile with ID details
2. Show available plots in preferred estate
3. Allocate plot with installment plan
4. Record initial deposit
5. Generate and send receipt
6. Explain payment schedule

#### Scenario 2: Payment Collection
**Situation:** Existing customer making installment payment.

**Tasks:**
1. Search for customer
2. View allocation and balance
3. Record payment
4. Generate receipt
5. Update balance
6. Check remaining installments

#### Scenario 3: Overdue Payment Follow-up
**Situation:** Customer has missed installment payment.

**Tasks:**
1. Generate overdue payments report
2. Identify customer and amount
3. Contact customer
4. Record payment when collected
5. Update status

### Assessment Quiz

#### For Frontdesk Staff

**Question 1:** What are the required fields when creating a customer profile?
a) Name and email
b) Name, phone, ID type, and ID number
c) Only name
d) Name and address

**Answer:** b

**Question 2:** How do you generate a receipt?
a) Manually create in Word
b) Automatically generated when payment is recorded
c) Download from external system
d) Request from IT

**Answer:** b

**Question 3:** What should you do if a payment was recorded incorrectly?
a) Delete the payment
b) Create a new payment with correct amount
c) Contact Admin to edit the payment
d) Ignore it

**Answer:** c

**Question 4:** When allocating a plot, what payment plans are available?
a) Only outright payment
b) Only installment
c) Outright or Installment
d) No payment plan needed

**Answer:** c

**Question 5:** How can customers receive their receipts?
a) Email only
b) SMS only
c) Download PDF or SMS
d) Printed copy only

**Answer:** c

#### For Agents

**Question 1:** Can you allocate plots using the mobile app?
a) Yes, same functionality as web
b) No, must use web only
c) Only view allocations
d) Only in online mode

**Answer:** a

**Question 2:** How are agent commissions calculated?
a) Manual calculation by admin
b) Automatic based on allocation amount
c) Fixed rate per plot
d) No commission tracking

**Answer:** b

**Question 3:** What happens when you record a payment offline?
a) Payment is lost
b) Payment syncs when internet is available
c) Must record again online
d) Payment is rejected

**Answer:** b
